Social Marketing
================

Small Odoo 16 example addon for managing social accounts and scheduled posts.

Features
--------

* Manage social publishing accounts.
* Draft, schedule, publish, or cancel posts.
* Track basic post metrics such as impressions and clicks.
* Provide a scheduled action hook for processing due posts.

Testing
-------

The repository test suite uses a lightweight mocked Odoo environment, so the
model logic can be validated without running a full Odoo server.
